> 安装

```
sudo yum install httpd -y 
```

> 服务

```
service httpd start 
service httpd restart
service httpd stop
```

> 查看状态

```
service httpd status
```

---

> web路径

```
/var/www/html/
```

> 配置文件路径

```
/etc/httpd/conf.d/welcome.conf
```
